# fullstack-interview-prep

# 🚀 Hành trình Ôn tập Phỏng vấn Fullstack Developer (2026)

Lộ trình này được thiết kế kéo dài 8 tuần, đi từ nền tảng Khoa học Máy tính cốt lõi (C++) cho đến thiết kế hệ thống và tích hợp Cloud (Angular & Firebase).

Mỗi giai đoạn dưới đây được quản lý trong một Repository riêng biệt. Bạn có thể bấm vào link từng giai đoạn để xem chi tiết code, tài liệu ghi chú và bài tập thực hành.

---

## 🗺️ Lộ trình Ôn tập

### 🟢 Giai đoạn 1: Nền tảng Khoa học Máy tính & Lập trình (Tuần 1-2)
**Mục tiêu:** Nắm chắc tư duy logic, cấu trúc dữ liệu và thuật toán, sử dụng C++ làm ngôn ngữ cốt lõi.

🔗 **[C++ & OOP & DSA](https://github.com/NH1Tkendo/01-cpp-and-dsa)**

- [ ] **Ngôn ngữ C++ & OOP:** Ôn tập cú pháp, con trỏ (pointers), quản lý bộ nhớ. Áp dụng 4 tính chất OOP (Đóng gói, Kế thừa, Đa hình, Trừu tượng).
- [ ] **Cấu trúc dữ liệu:** Array, Linked List, Stack, Queue, Tree (BST), Graph, Hash Table.
- [ ] **Thuật toán:** Quick sort, Merge sort, Binary search, Dynamic Programming, Greedy, DFS/BFS. Phân tích độ phức tạp Big-O.

### 🟡 Giai đoạn 2: Cơ sở dữ liệu (Tuần 3)
**Mục tiêu:** Thao tác mượt mà với dữ liệu, hiểu cách thiết kế và tối ưu database.

🔗 **[Truy cập Repository Giai đoạn 2 tại đây](LINK_REPO_GIAI_DOAN_2_CUA_BAN)**

- [ ] **SQL:** Truy vấn nâng cao (JOIN, GROUP BY, HAVING, Subqueries). Hiểu về ACID, Transaction, Chuẩn hóa dữ liệu (Normalization). Tối ưu truy vấn bằng Indexing.
- [ ] **NoSQL:** Phân biệt SQL và NoSQL. Thao tác với cấu trúc dữ liệu dạng Document.

### 🟠 Giai đoạn 3: Phát triển Backend & API (Tuần 4)
**Mục tiêu:** Hiểu cách server hoạt động và giao tiếp với client.

🔗 **[Truy cập Repository Giai đoạn 3 tại đây](LINK_REPO_GIAI_DOAN_3_CUA_BAN)**

- [ ] **Mạng & Giao thức:** Mô hình OSI, TCP/IP, HTTP/HTTPS, HTTP Methods, Status Codes.
- [ ] **Xây dựng API:** Thiết kế RESTful API chuẩn mực.
- [ ] **Bảo mật & Tối ưu:** Authentication/Authorization (JWT, OAuth2), Xử lý lỗi, Middleware, Logging.

### 🔵 Giai đoạn 4: Phát triển Frontend với Angular (Tuần 5)
**Mục tiêu:** Xây dựng giao diện người dùng tương tác và tối ưu.

🔗 **[Truy cập Repository Giai đoạn 4 tại đây](LINK_REPO_GIAI_DOAN_4_CUA_BAN)**

- [ ] **Core Web:** HTML5, CSS3 (Flexbox, Grid), JavaScript (ES6+, Event Loop, Promises, DOM).
- [ ] **Angular Framework:** Component Lifecycle, State Management, Dependency Injection, Services.
- [ ] **RxJS & Routing:** Xử lý bất đồng bộ (Observables, Subjects), Reactive Forms.

### 🟣 Giai đoạn 5: Tích hợp Fullstack & Môi trường Cloud (Tuần 6)
**Mục tiêu:** Ghép nối Frontend và Backend thành một sản phẩm hoàn chỉnh, sẵn sàng triển khai.

🔗 **[Truy cập Repository Giai đoạn 5 tại đây](LINK_REPO_GIAI_DOAN_5_CUA_BAN)**

- [ ] **Tích hợp:** Kết nối Angular với Backend/BaaS để xử lý Realtime Database, Authentication.
- [ ] **DevOps Cơ bản:** Git workflows (Branching, Merge conflict, Rebase), CI/CD khái quát, Docker cơ bản.

### ⚫ Giai đoạn 6: Thiết kế Hệ thống & Portfolio (Tuần 7-8)
**Mục tiêu:** Vượt qua các câu hỏi phỏng vấn cấp cao và chứng minh năng lực thực tế.

🔗 **[Truy cập Repository Giai đoạn 6 tại đây](LINK_REPO_GIAI_DOAN_6_CUA_BAN)**

- [ ] **System Design:** Scale hệ thống (Vertical/Horizontal), Load balancing, Caching. Microservices vs Monolithic.
- [ ] **Trình bày Portfolio:** Trình bày kiến trúc các dự án phức tạp đã làm (xử lý luồng dữ liệu, tích hợp thuật toán gợi ý/recommendation algorithms, thiết kế pipeline tiền xử lý dữ liệu...).
- [ ] **Phỏng vấn Hành vi:** Chuẩn bị câu trả lời theo mô hình STAR.
